# IbosonTechStore – E-Commerce App (React + FakeStoreAPI)

A **modern, responsive e-commerce store** with **real-time cart**, **search**, **filters**, **product modal**, and **persistent cart**.

---

## Features

- **Real-time Cart** (add, remove, update quantity, clear)
- **Header Cart Badge** – Updates instantly
- **Search Bar** with debounce
- **Sidebar Filters** (Category in-stock, price range, category)
- **Price Range Slider** (purple track, labels)
- **Product Grid** (3-col → 2-col → 1-col responsive)
- **Product Modal** with related products
- **Cart Drawer** (slide-in, beautiful UI)
- **LocalStorage Persistence** – Cart survives refresh


---

## Tech Stack

- **React 18**
- **React Icons** (`Fi*`)
- **react-range** (price slider)
- **react-responsive** (mobile detection)
- **FakeStoreAPI** (real product data)
- **LocalStorage** (cart persistence)
- **CSS Modules** (scoped styles)
